8 bit PC

Ovaj projekt je moj vlastiti dizajn 8-bitnog procesora i računala u programu Logisim.
Dizajn procesora je u potpunosti samostalno napravljen, te ne koristi arhitekturu nijednog meni poznatog procesora.
Računalo podržava 81 različite instrukcije i sposobno je izvoditi programe poput 
rastavljanja broja na proste faktore i traženja prostih brojeva. 
Računalo je Turing kompletno.
Računalo radi s 32-bitnim podatcima, sposobno je računati i upravljati brojevima od 0 do 4,294,967,295. 
Za adresiranje koristi 24 bita, što znači da može adresirati do 16,777,215 32-bitnih registara RAM memorije.
Takva RAM memorija ima 64 MB memorije.
Uz RAM memoriju također postoje procesorski registri R0,R1,R2,R3 i A. 
Procesor koristi 8-bitne instrukcije, od kojih je 19 dizajnirano za upravljanje memorijom, 
46 za računanje, 15 za skokove i 1 za označavanje završetka programa. 
Maksimalna brzina rada procesora je 4.1 KHz.

Primjer programa koji procesor izvodi:

Zadatak programa: Ispisivati proste brojeve
Ispis (HEX): 2, 3, 5, 7, B, D, 11, 13, 17, 1D, 1F, 25, 29, 2B, 2F, 35, 3B, 3D, 43, 47, 49…

Assembly program:
// Komentari:
// var count - broj dijeljenja kada je ostatak = 0
// var n - broj za koji se provjerava je li prost
// var m - broj [1,n] za provjeru
// var adr - broj koji prati adresu memorije za zapis prostih n
// var ADR1 = 07
// var ADR2 = 14
// var ADR3 = 23

MOV *01,A
MOV A,R2 // const za usporedbu count s 1

MOV *04,A
MOV A,R3 // adr = 4 (početna vrijednost)

ADR1:
MOV *01,A
MOV A,01 // m = 1

MOV 00,A
INC A
MOV A,00 // INC n

MOV *00,A
MOV A,02 // count = 0

ADR2:
MOV 01,A
MOV A,R0 // 01 > R0
MOV 00,A
MOD A,R0 // n % m > A

JMP A NE 0, ADR3 // preskakanje INC count

MOV 02,A
INC A ; 02 + *1 > A
JMP A GT R2, ADR1 // prekidanje provjere n ako count > 1
MOV A,02 // INC count

ADR3:
MOV 00,A
MOV A,R1 // n > R1 za usporedbu s m

MOV 01,A
INC A
MOV A,01 // INC m

JMP A LT R1, ADR2 // nastavlja se rad s idućim m

MOV 00,A
MOV A,*R3 // Spremanje prostog broja na adr
MOV R3,A
INC A
MOV A,R3 // INC adr
JMP ADR1 // provjera idućeg n
